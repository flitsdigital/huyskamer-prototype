A dish row from the menukaart — name + price baseline with optional description.

```jsx
<MenuItem name="Caesar salade" price={10.25}
  description="Met gerookte kip reepjes, uitgebakken spekjes en ei" />
<MenuItem name="Tomatensoep" price={4.75} vegetarian />
```

Group rows under a script category heading (use the large script type). `price` accepts a number (auto "€ 7,50") or string.
