One-colour line icon on a 24×24 grid — the brand's Relume set, substituted with visually-identical Lucide line glyphs (stroke 2, round joins). Inherits `currentColor`.

```jsx
<Icon name="map-pin" size={20} />
<span style={{ color: "var(--brand)" }}><Icon name="phone" /></span>
```

Available names: menu, close, arrow-right, chevron-down, chevron-right, phone, mail, map-pin, clock, info, check, check-circle, star, instagram, facebook, whatsapp. `star`, `whatsapp`, `facebook`, `instagram` render solid; the rest are strokes.
